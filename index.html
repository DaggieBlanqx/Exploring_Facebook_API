Project Introduction

This project was initiated to mostly explore the Graph API provided by Facebook.Python 2.7 is used to get access to JSON data provided by the API. So lets get started.

Getting the packages

Install the necessary python pckages ,either using 

pip install packagename
easy_install packagename

or download the package and try 

setup install
